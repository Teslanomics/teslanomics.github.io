# Teslanomics

> Decoding the Economics of Tesla

# FOSS
Hosted on **Netlify** with **Forestry.io**

## To get started

```sh
$ gulp [--prod]
```

## Usage

```sh
$ gulp fullbuild [--prod]
```


#### Settings
In your `_config.yml` and `humans.txt` you should add your Github and Twitter
profile if you want to.

## Install
```sh
$ yarn setup
```
